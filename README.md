# dotfiles-claude

Personal [Claude Code](https://claude.ai/code) configuration.

## Contents

| File | Description |
|------|-------------|
| `CLAUDE.md` | Global instructions and preferences for Claude |
| `settings.json` | MCP servers, hooks, permissions, env vars, status line |
| `policy-limits.json` | Policy restrictions |
| `statusline-command.sh` | Custom status line command |
| `skills/` | Custom slash command skills |

## Setup

```bash
# Clone into ~/.claude
git clone git@github.com:brydoncheyney/dotfiles-claude.git ~/.claude
```
